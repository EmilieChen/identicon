  Our Identicons are 5Ã—5 sprites of 50 pixels. They are generated using a hash of the input string as the seed. The algorithm walks through the hash and colors sprites depending on even or odd values of the hash. The colors is determined by the first three values of the hash. The algorithm ensures a huge number of unique 250 x 250 resolutions identicons.
